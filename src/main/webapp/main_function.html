02 강제종료 차단 설정

강제종료 차단 설정
사용자가 임의로 클라이언트를 종료하거나 제어판에서 프로그램을 삭제하지 못하도록 관리자가
인증 비밀번호를 설정하여 사용자의 클라이언트 강제종료 및 삭제를 차단 할 수 있습니다.

강제종료 차단 설정 방법

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 삭제 차단 설정 “예” 체크 후 관리자만이 알 수 있는 인증번호를 입력합니다.

3 설정 저장 버튼을 클릭 합니다.

강제종료 차단 설정

4 사용자가 클라이언트 종료버튼 클릭 및 제어판 삭제 시 인증 비밀번호 입력 창이 생성되며,
관리자가 생성한 인증 비밀번호를 입력하게 되면 클라이언트를 강제 종료할 수 있습니다.

03 복호화 권한 설정

복호화 권한설정
암호화된 파일에 관한 복호화 권한 정책으로 개인정보 취급자가 아닌 일반사용자가 암호화된 파일을 임의로 복호화 할 수 없도록 관리자가 권한을 설정 할 수 있습니다.

암호화된 파일의 복호화 권한 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 암호화 된 파일의 복호화 권한 설정 “아니오”를 선택 후 설정 저장버튼을 클릭 합니다.

결제창 팝업

3 복호화 권한이 없는 사용자가 암호화 된 파일을 복호화 하려고 할 때 위와 같은 메시지 창이 나타나며, 복호화를 차단 합니다.

4 복호화 권한이 없는 경우 반드시 복호화 결재 프로세스를 사용하여 결재 후 복호화를 진행 하셔야 합니다.

복호화 사유

5 복호화 권한이 있는 사용자의 경우 복호화 이유를 상세히 기록한 후 복호화를 진행 합니다.

복호화 내용 목록

6 복호화 사유 내용을 서버에서 확인 할 수 있습니다.

04 복사 방지 설정

복사방지 권한설정
파일에 존재 하는 내용(TXT)을 비보안 대상에 복사 할 수 없도록 설정하는 기능 입니다.

암호화 파일 내용 복사 방지 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 복사 방지 설정 “예” 선택 후 설정저장 버튼을 클릭 합니다.

3 파일의 내용을 복사 허용 어플리케이션(파워포인트, 엑셀, 한글, 워드, 알씨, 그림판, 한쇼, 한셀)이 아닌 어플리케이션으로 내용(TXT)복사 시 복사를 허용하지 않습니다.

복사방지 화면

4 복사 방지 기능이 실행 될 경우 위와 같은 보안 메시지를 텍스트로 보여줍니다.

05 실시간 감시 설정

실시간 감시기능 강제사용 설정
클라이언트의 실시간 감시기능 설정을 관리자가 강제로 제어 할 수 있는 기능으로
관리자가 실시간 감시 강제사용을 선택할 경우 클라이언트에서 사용자가 임의로 실시간 감시를 해지할 수 없습니다.

실시간 감시기능 강제사용 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 실시간 감시기능 강제 설정 “예” 선택 후 설정저장 버튼을 클릭 합니다.

실시간 감시기능 강제사용 설정

3 실시간 감시 설정 변경 후 SPK 재 시작 시 실시간 감시 기능이 강제적으로 “ON”으로 변경됩니다.

06 비밀번호 유효기간 설정

비밀번호 유효기간 설정
클라이언트 사용자의 비밀번호 유효기간을 설정할 수 있는 기능입니다.

비밀번호 유효기간 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 유효기간 설정 “예” 선택 및 변경 주기 일수를 입력 후 설정저장 버튼을 클릭 합니다.

3 최종 비밀번호 변경일로부터 설정된 일자가 되면 사용자 비밀번호 변경팝업이 노출됩니다.

비밀번호 유효기간 설정 팝업

4 노출된 팝업 클릭 시 아래와 같이 비밀번호 변경 화면으로 자동 이동됩니다

비밀번호 변경

4 현재 비밀번호와 새 비밀번호 입력 후 설정 저장 클릭

07 검사 대상 제외 경로 설정

검사 대상 제외 경로 설정
검사제외 대상을 지정하여 검사 시 해당 경로에 있는 파일은 검사를 진행 하지 않도록 설정할 수 있는 기능입니다.

검사 대상 제외 경로 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 폴더 경로를 입력하여 경로추가 버튼을 클릭 후 설정저장 버튼을 클릭 합니다.

3 검사 시 등록된 경로는 개인정보가 존재하여도 검출되지 않습니다.

4 검출 파일 처리 유형이 암호화 또는 삭제로 설정 되어 있어도, 등록된 경로의 파일들은 개인정보 검출이 되지 않아 처리 유형 설정 값이 적용되지 않습니다.

08 패턴설정

패턴 설정
필요한 패턴정보를 추가 설정하여 검사 시 다양한 개인정보를 검출을 할 수 있도록 설정하는 기능입니다.

패턴 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 클라이언트에 적용할 개인정보 검출 패턴을 등록합니다. 주민등록번호(기본)와 외국인 등록번호(기본)는 기본 값으로 설정되어 있습니다.

3 각 패턴 항목에서 설정한 처리 활성화 수 이상의 검출 수만 검출됩니다. (초기 값은 1로 설정되어 있습니다.)

패턴 설정

4 관리자 정책으로 설정한 패턴은 사용자 임의로 제거 할 수 없습니다.

09 출력 제어 설정

출력 제어 설정
출력 페이지량 제한 또는 출력 시 개인정보 마스킹을 할 수 있도록 설정할 수 있는 기능입니다.

출력 제어 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 출력제어 기능 설정 “예” 선택 후 설정저장 버튼을 클릭 합니다.

3 출력제어 설정을 하지 않으면 출력 사용량 설정, 마스킹 설정이 되어 있어도 설정된 기능이 동작하지 않습니다.

출력 사용량 설정

1 일별, 월별 출력 가능한 페이지를 설정하여 설정된 페이지를 초과하면 인쇄 가능 페이지 초과 메시지와 함께 출력이 되지 않습니다.

출력 사용량 설정

개인정보 마스킹 설정

1 출력 시 개인정보(주민번호)를 검출하여 부분 마스킹(뒤 7자리전체 또는 성별자리 제외 마스킹), 전체 마스킹을 할 수 있습니다.

12 네트워크서비스 제어

제어 프로그램 설정

13 시스템 제어 설정

시스템 제어
클라이언트가 동작하는 PC의 시스템 비밀번호, 비밀번호 유효기간, 화면 보호기 설정을 제어할 수 있습니다.
해당 설정 적용 후 해제는 반드시 해당 화면에서 해제하셔야 합니다.

시스템 비밀번호 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 시스템 비밀번호 유효기간 설정 “예”를 선택 합니다.

3 윈도우 시스템 비밀번호 설정 시 비밀번호의 최대/최소 길이를 설정 후 설정저장을 클릭 합니다.

시스템 비밀번호 유효기간 설정정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 시스템 비밀번호 유효기간 설정 “예”를 선택 합니다.

3 비밀번호 유효기간을 선택 후 설정저장 버튼을 클릭 합니다.

4 설정된 비밀번호 유효기간 초과 시 윈도우 비밀번호 변경 화면이 노출되며, 비밀번호를 변경하여야 합니다.

시스템 화면보호기 설정

1 좌측 조직 구성도에서 설정 변경 할 사업장/부서/사용자를 선택 합니다.

2 보호기 설정에서 “예”를 선택 합니다.

3 활성화 대기 시간을 선택 후 설정저장 버튼을 클릭 합니다.

4 화면보호기가 윈도우에 설정되어 있지 않아도 해당 설정으로 강제 적용합니다.

비밀번호 및 유효기간 설정
SpkOne 계정 생성

5 시스템 제어 설정을 적용 하면 SpkOne 계정이 생성 되며
사용자가 자신의 윈도우 로그인 비밀번호를 잊어버렸을 경우
SpkOne 계정으로 로그인 후 사용자 비밀번호를 초기화 합니다.
