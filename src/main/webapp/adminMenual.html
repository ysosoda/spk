01. 제품 개요

  • 메디세이퍼는 보안의 핵심인 개인정보보호, 문서보안, 출력보안, Email보안, 네트워크보안, 랜섬웨어 피해방지를 하나의 Agent로 중요정보의 내부유출을 방지하는 통합정보보호 솔루션입니다.
  • 저희 솔루션은 정보의 생성부터 원천 암호화하여 높은 수준의 보안성을 확보하였으며, 사용자 편의성을 높이고 관리포인트와 리소스를 최소화 하였습니다.
  • 저희 솔루션은 클라이언트 환경에서 수집하는 Client 프로그램과 모든 클라이언트의 관리 및 검사, 검사 결과 등을 조회 및 보고를 담당하는 웹 환경의 Server 관리프로그램으로 구성되어 있습니다.

02. 관리자 사이트 접속

  • 메디세이퍼 관리자시스템은 웹 환경으로 제공하고 있으며 Internet explorer 9.0 이상의 브라우저를 필요로 합니다. 관리시스템에 접속하기 위해서는 서버시스템 설치 시 설정된 IP와 Port를 이용하여 웹 주소 입력 창에 입력 후 접속을 합니다.

  • 로그인은 관리시스템을 운영하는 관리자가 적합한지 확인하는 절차로 모든 클라이언트로부터 수집된 정보를 보존/조회/리포팅하기 위한 사용자 검증 단계입니다.

03. DashBoard

로그인 후 첫 화면으로 관리자가 현재 관리중인 조직의 설치 정보, 보안 상태 등을 한눈에 볼 수 있습니다.
  • 버전정보: 제품로고 및 현재 설치 된 Server 버전을 나타냅니다. 제품로고 클릭 시 Dashboard 화면을 보여줍니다.
  • 라이선스 정보 : 등록된 라이선스 만기일을 표시합니다.
  • 관리자정보: 현재 서버에 접속중인 관리자의 정보와 로그아웃 단추를 제공합니다.
  • 기간별 검사 현황: 조직 내 사용자의 검사건수를 일별(최근1주일), 월별(최근6개월)로 나누어 검사유형별 결과를 표시해 줍니다.

  • 패턴별 검출 현황: 조직 내 사용자의 검출건수를 오늘, 최근1주일, 최근1개월로 나누어 개인정보를 포함하고 있는 패턴별 검출 결과로 표시해 줍니다.
